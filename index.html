Born in 1990, Taoyuan, Taiwan, Yi-Hsien Hung is a graphic designer, visual artist, illustrator based in the Netherlands.

Yi-Hsien is experienced with illustration, watercolour painting, visual identity, packaging design, editorial design, typography, web design, user interface design.
